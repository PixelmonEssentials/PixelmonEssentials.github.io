Getting Started



Pixelmon Essentials requires the following mods to be installed to work:

  • Pixelmon Reforged (Any 1.12.2 version should work)
  • Custom NPCs (upcoming snapshot needed)

To work correctly, the mod needs to be installed on the server, if it's used on a server, and will need to be installed on any client playing for an adventure map. If a world is opened to LAN to play the map in multiplayer, only the host needs to have it installed.

Multiple times in the tutorials, there might be 2 folders mentioned for different reasons: the essentials folder, and the data folder. The essentials folder can be found in %world folder%/pixelmonessentials/ while the data folder can be found in %world folder%/pixelmonessentials/data/

If you currently have access to a Pixelmon Esssentials build, please do not redistribute it! It's still a work in progress, and as such has been given limited distribution for the time being.
